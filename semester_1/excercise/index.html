## Bad Piggies Use what you have learned about CSS specificity to change color of the piggies. 1. Add CSS that uses the lowest specificity possible to override the `background-color` of each piggy without changing the HTML. 2. Each piggies should be a specific color as follows: 1) `blue`, 2) `green`, 3) `orange`, 4) `purple`, 5) `pink`, 6) `grey`, 7) `white`, 8) `gold`.
Example for piggy 1:
`main div { background-color: blue }`
