Benefits

Functional Prototyping

Bootstrap cross quicker from the sketch to a realistic and functional Prototype, giving you a powerful tool to validate early user insights.

Velocity

Based on standard components, it is possible to assemble in a short time a complete experience for websites, landings or any digital interface.

Mobile First

The model has a mature grid system, consequently the structure and modules, giving you the posibility to display the interface for any range of size.

Flexbox Based

Bootstrap is based on Flexbox; workin with flexible containers allows you to manipulate the grid and the components inside them in versatile way.

Personalization

You can make sophisticated customizations; even though the framework has standard components, you can make many adjustments.

Easy to put into practice

As far you have the main concepts of HTML and CSS, Bootstrap will be easy to onboard and learn; besides, there is a lot of information around this framework.

Accordions

This is the first item's accordion body. It is hidden by default, until the collapse plugin adds the appropriate classes that we use to style each element. These classes control the overall appearance, as well as the showing and hiding via CSS transitions. You can modify any of this with custom CSS or overriding our default variables. It's also worth noting that just about any HTML can go within the .accordion-body, though the transition does limit overflow.
